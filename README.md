# safeinbox-phishing-email-detector 
AI-powered Chrome extension for real-time phishing email detection in Gmail.

## Features
- Real-time phishing email detection in Gmail
- Automatic risk score prediction
- Suspicious link highlighting
- Warning banner for unsafe emails
- Machine learning model with 97.05% accuracy
- High phishing detection recall (98%)

## Machine Learning
The phishing detection model is trained using:

- TF-IDF Vectorization
- Logistic Regression
- Scikit-learn

### Model Performance
- Accuracy:  97.05%
- Precision: 95%
- Recall:    98%
- F1-score:  96%

## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML
- CSS
- JavaScript
- Chrome Extension

## Use Case
SafeInbox helps users identify phishing emails in Gmail before clicking malicious links or sharing sensitive credentials.
